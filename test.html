                <h2><i class="material-icons text-blue-600 mr-2 align-bottom">info</i>研究背景</h2>

                <p class="mb-3">沙门氏菌（Salmonella）是引发食源性胃肠道感染的主要病原体之一，常导致急性腹泻、肠道炎症，并可能引发全身性感染。除了消化道症状，越来越多的研究表明，肠道感染，特别是沙门氏菌感染，与焦虑、抑郁等情绪障碍和认知功能缺陷有关。这种联系凸显了“肠-脑轴”（Gut-Brain Axis）的重要性，即肠道和大脑之间复杂的双向通讯系统。</p>

                <p class="mb-3">迷走神经（Vagus Nerve）作为肠-脑轴的关键物理连接，传递来自肠道的多种信号（如激素、神经递质、炎症信号）到大脑，同时也向肠道传递信号调节肠道功能和免疫反应。肠道微生物群及其产生的代谢物（如γ-氨基丁酸，GABA）在调节宿主免疫、维持肠道屏障完整性和影响神经功能方面也发挥着至关重要的作用。沙门氏菌感染会破坏肠道微生物稳态，可能影响GABA等关键代谢物的水平。</p>

                <div class="highlight">

                  <p><strong>核心问题：</strong>沙门氏菌感染引发焦虑样行为的具体机制尚不完全清楚，特别是迷走神经和肠道微生物（尤其是GABA）在其中扮演的角色有待阐明。</p>

                </div>

            </section>


            <section class="card">

                 <h2><i class="material-icons text-blue-600 mr-2 align-bottom">track_changes</i>研究目的与假设</h2>

                 <p>本研究旨在探讨迷走神经和肠道神经递质（特别是GABA）在调节鼠伤寒沙门氏菌（*S. typhimurium*）诱导的全身感染、神经炎症反应和焦虑样行为中的作用。</p>

                 <p class="mt-2"><strong>主要假设：</strong>切断迷走神经（迷走神经切断术，Vagotomy）能够通过影响肠道微生物群及其代谢产物（如GABA），减轻沙门氏菌感染引起的肠道炎症和相关的焦虑样行为改变。</p>

            </section>


             <section class="card">

                <h2><i class="material-icons text-blue-600 mr-2 align-bottom">science</i>研究思路与方法</h2>

                <p class="mb-4">本研究采用小鼠模型，通过外科手术（迷走神经切断术）和药物干预（GABA补充），结合行为学测试、分子生物学、组织病理学、微生物组学和代谢组学等方法，系统地探究了迷走神经和GABA在沙门氏菌感染中的作用。</p>


                <h3 class="text-lg font-semibold mb-2 text-gray-700">主要实验设计:</h3>

                <ul class="list-disc list-inside space-y-1 mb-4">

                    <li><strong>实验一：</strong>比较四组C57BL/6J雄性小鼠：假手术组（Sham）、假手术+沙门氏菌感染组（Sham+S.typhimurium）、迷走神经切断组（Vagotomy）、迷走神经切断+沙门氏菌感染组（Vagotomy+S.typhimurium）。</li>

                    <li><strong>实验二：</strong>比较两组小鼠：沙门氏菌感染组（S.typhimurium）、沙门氏菌感染+GABA预处理组（S.typhimurium+GABA）。</li>

                </ul>


                 <h3 class="text-lg font-semibold mb-2 text-gray-700">关键技术与指标:</h3>

                 <ul class="list-disc list-inside space-y-1 mb-4">

                     <li><strong>迷走神经切断术:</strong> 单侧颈部迷走神经切断。</li>

                     <li><strong>沙门氏菌感染:</strong> 经口灌胃给予1x10⁹ CFU的*S. typhimurium* SL1344。</li>

                     <li><strong>行为学评估:</strong> 旷场实验（Open Field Test, OFT）、埋珠实验（Marble Burying Test, MBT）评估焦虑样行为。</li>

                     <li><strong>感染与炎症指标:</strong> 体重变化、脾脏指数、脾脏沙门氏菌载量、血清脂多糖（LPS）水平、肠道组织H&E染色、Alcian Blue染色（杯状细胞）、肠道和大脑相关细胞因子（TNF-α, IL-1β, IL-6, IL-10, IL-22, CXCL1）mRNA表达（RT-PCR）。</li>

                     <li><strong>神经炎症与活动:</strong> 大脑（海马、杏仁核、皮层）小胶质细胞活化（IBA-1免疫荧光）、神经元活化（c-Fos免疫荧光）。</li>

                     <li><strong>肠道神经递质:</strong> 结肠内容物中GABA、组胺、谷氨酸、色氨酸、N-乙酰血清素水平（LC-MS/MS）。</li>

                     <li><strong>肠道微生物组:</strong> 粪便样本16S rRNA测序分析α/β多样性、菌群组成。</li>

                </ul>

                <h2><i class="material-icons text-blue-600 mr-2 align-bottom">assessment</i>关键结果</h2>

                <p class="mb-4">研究结果清晰地揭示了迷走神经切断术和GABA干预对沙门氏菌感染引起的一系列病理生理和行为改变具有显著的缓解作用。</p>


                <div class="grid grid-cols-1 md:grid-cols-2 gap-6">

                    <div>

                        <h3 class="text-lg font-semibold mb-2 text-indigo-700">迷走神经切断术 (Vagotomy) 的影响:</h3>

                        <ul class="space-y-1">

                           <li class="list-item">

                               <span class="material-icons list-icon">healing</span>

                               <span><strong>减轻全身感染：</strong>显著减轻了感染引起的体重下降、脾脏肿大、脾内沙门氏菌负荷及血清LPS水平升高。</span>

                           </li>

                           <li class="list-item">

                                <span class="material-icons list-icon">psychology</span>

                               <span><strong>缓解焦虑样行为：</strong>改善了旷场实验中的探索行为（中央区域活动增加），并减少了埋珠行为。</span>

                           </li>

                           <li class="list-item">

                               <span class="material-icons list-icon">local_hospital</span>

                               <span><strong>抑制肠道炎症：</strong>减轻了空肠和结肠的组织损伤（炎症细胞浸润、绒毛/隐窝破坏减少），保护了杯状细胞，并下调了促炎因子（TNF-α, IL-1β, IL-6, IL-10）表达，同时恢复了抗感染因子（IL-22, CXCL1）水平。</span>

                           </li>

                            <li class="list-item">

                               <span class="material-icons list-icon">neurology</span>

                               <span><strong>降低神经炎症：</strong>减少了海马和杏仁核中的小胶质细胞活化（IBA-1），并降低了海马和皮层的促炎因子（IL-6, IL-1β, TNF-α, IL-10）水平。</span>

                           </li>

                           <li class="list-item">

                               <span class="material-icons list-icon">electrical_services</span>

                               <span><strong>阻断神经信号：</strong>降低了感染诱导的孤束核（NTS）和杏仁核中c-Fos（神经元活化标记物）的表达。</span>

                           </li>

                            <li class="list-item">

                               <span class="material-icons list-icon">mediation</span>

                               <span><strong>调节肠道神经递质：</strong>逆转了感染导致的结肠GABA、色氨酸和N-乙酰血清素水平下降。感染引起的谷氨酸和组胺水平升高则不受迷走神经切断术影响。</span>

                           </li>

                            <li class="list-item">

                 <h2><i class="material-icons text-blue-600 mr-2 align-bottom">speaker_notes</i>讨论与结论</h2>

                 <p class="mb-3">本研究证实，沙门氏菌感染可通过激活迷走神经通路，引起宿主中枢神经系统的激活（NTS、杏仁核）和局部炎症（海马、杏仁核），最终导致焦虑样行为。迷走神经似乎是传递肠道感染“警报”信号至大脑的关键途径，可能感知肠道释放的炎症因子（如IL-1β, TNF-α）或神经递质（如谷氨酸，组胺）的变化。</p>

                 <p class="mb-3">迷走神经切断术不仅阻断了这种自下而上的致焦虑信号，还显著改善了肠道微环境。它抑制了肠道炎症，并重塑了肠道菌群结构，特别是增加了产生GABA的有益菌丰度，恢复了结肠GABA水平。这表明迷走神经信号通路可能参与调节肠道菌群稳态，或者感染诱导的焦虑状态本身会破坏菌群稳态，而切断迷走神经缓解了焦虑，间接保护了菌群。</p>

                 <p class="mb-3">GABA补充实验进一步证实了其保护作用。口服GABA能有效减轻沙门氏菌引起的全身感染、肠道和大脑炎症以及焦虑样行为，这与迷走神经切断术的效果高度相似。GABA可能通过抑制炎症反应、保护肠道屏障和直接的神经调节作用来发挥其抗感染和抗焦虑效果。</p>

                  <div class="highlight bg-green-100 border-green-500">

                    <p><strong>结论：</strong>迷走神经在沙门氏菌感染引起的肠-脑通讯和焦虑行为中扮演核心角色。干预迷走神经通路或补充其下游关键调节因子GABA，均能有效缓解感染带来的多系统损害。靶向“迷走神经-肠道菌群-GABA”轴可能为治疗细菌感染及其相关的神经精神并发症提供新的策略。</p>

                 </div>

                 <h2><i class="material-icons text-blue-600 mr-2 align-bottom">report_problem</i>局限性</h2>

                  <ul class="list-disc list-inside space-y-1">

                     <li>迷走神经切断术影响肠道菌群的具体机制（是直接影响肠道环境还是通过中枢信号改变）仍需深入研究。</li>

                     <li>沙门氏菌激活迷走神经的确切分子机制（是LPS、细胞因子、代谢物还是直接作用于神经末梢）有待进一步通过体外共培养等方法探明。</li>

                     <li>沙门氏菌诱导焦虑所涉及的详细中枢神经环路图谱需要利用更先进的神经示踪和调控技术进行解析。</li>

                     <li>GABA发挥抗感染作用的具体途径（是直接抑制细菌还是调节宿主免疫/屏障功能）需要进一步区分。</li>
